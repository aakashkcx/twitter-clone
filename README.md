# Twitter Clone

## Installation

Clone the git repository:

```shell
$ git clone https://github.com/aakashkcx/twitter-clone.git
```

Install dependencies with npm:

```shell
$ npm run install
```

Start the server:

```shell
$ npm run sever
```

or `npm run sever-dev`

Start the client:

```shell
$ npm run client
```
